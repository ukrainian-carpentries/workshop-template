Оболонка Bash

Bash — це широко використовувана оболонка, яка дає вам можливість виконувати задачі швидше.

  • Windows
  • MacOS
  • Linux
  1. Завантажте інсталятор Git для Windows.
  2. Запустіть інсталятор і виконайте наступні дії:

Це забезпечить найвищий рівень сумісності наших уроків. {% comment %} У цьому розділі також є "Override the default branch name for new repositories" (Перевизначити типову назву гілки для нових репозиторіїв) і текстове поле встановлено на "main" (Головне).     4. Переконайтеся, що вибрано "Git from the command line and also from 3rd-party software" (Git з командного рядка, а також з програмного забезпечення 3-ї сторони) та натисніть на "Next" (Далі).     10. Переконайтеся, що вибрано "Git Credential Manager" (Менеджер облікових даних Git), і натисніть "Next" (Далі).
    11. Переконайтеся, що вибрано "Enable file system caching" (Увімкнення кешування файлової системи), і натисніть "Next" (Далі).

    12. Click on "Install" (Встановити).
      13. {% comment %} Встановлення {% endcomment %} {% comment %} Завершення роботи Git Setup Wizard (Майстра налаштування Git) {% endcomment %} {% comment %} станом на 02.06.2020 у вікні буде написано "click Finish" (Готово), але кнопка позначена як "Next" (Далі) {% endcomment %}
    13. Натисніть на "Finish" (Готово) або "Next" (Далі).
  3. Якщо вашу змінну середовища "HOME" (ГОЛОВНА СТОРІНКА) не встановлено (або ви не знаєте, що це таке):
    1. Відкрийте командний рядок (відкрийте Start Menu (меню "Пуск"), потім введіть cmd і натисніть Enter)
    2. Введіть наступний рядок у вікно командного рядка точно так, як показано:

      setx HOME "%USERPROFILE%"

    3. Натисніть Enter і ви повинні побачити SUCCESS: Specified value was saved.
    4. Вийдіть із командного рядка, ввівши exit потім натиснувши Enter

Це надасть вам як Git, так і Bash у програмі Git Bash.

Відео-інструкція

Оболонка за замовчуванням в Mac OS X Ventura і більш нових версіях - є Zsh, але Bash доступний у всіх версіях, тому встановлювати нічого не потрібно. Доступ до Bash можна отримати з терміналу (знаходиться в /Applications/Utilities). Перегляньте відео-інструкцію інсталяції Git для прикладу того, як відкрити Термінал. Можливо, ви захочете закріпити Термінал на для цього семінару.

Щоб перевірити, чи є ваша типова оболонка Bash, введіть echo $SHELL у Терміналі та натисніть клавішу Return. Якщо виведене повідомлення не завершується на '/bash' , вашою типовою оболонкою є щось інше. Ви можете змінити вашу поточну командну оболонку на Bash введенням bash і натисканням Return. Щоб перевірити поточну командну оболонку, введіть echo $0 і натисніть Return.

Щоб змінити типову оболонку на Bash, введіть chsh -s /bin/bash і натисніть клавішу Return, а потім перезавантажтеся, щоб зміни набули чинності. Щоб змінити типове значення назад на Zsh, введіть chsh -s /bin/zsh, натисніть клавішу Return і перезавантажтеся. Щоб перевірити доступні оболонки, введіть cat /etc/shells.
